Colors in CSS

  • About 147 different "named" colors to choose from
  • Hexadecimal Colors
    • The first 2 places (left to right) corresponds to how much red is in the color
    • The next 2 places corresponds to how much green is in the color
    • The last 2 places corresponds to how much blue is in the color
    • Search online for the hexadecimal value color picker
  • RGB - 3 channels: Red, Green and Blue. Each Range from 0-255
    • An example would be color: rgb(0,255,255); This would be the same in hex as #00FFFF
  • RGBA - Just like RGB, but with an aplha(transparency) channel. Ranges from 0.0-1.0
    • An example of RGBA would be color: rgba(0,255,255,.2)

Color and Background

Use 'color' to set text color and 'background' for background color

Background Image

The background property can also set a background iamge

Borders

In CSS, you can add borders around elements

  • border-color: purple;
  • border-width: 5px;
  • border-style: solid,dashed, etc;
  • You can use a shortcut using: border: 5px, solid, purple

Specificity

Specificity - idea that we can have multiple styles targeting one element. Which ever is more specific usually wins. There are weights that determines which styles would win over other styles

  • Elements and Pseduo-Elements (such as p, body, ul, etc): 1
  • Classes, attributes, and Pseduo-Elements (such as .whatever, input[type="text"]): 10
  • IDs (such as #whatever: 100

Text Manipulation - Fonts/Lines/Text

Font-Family - Changes the type of font with the font-family command

Font-Size - Changes the font size with font-size command

There's another way to change the font size with the em value. The size of an em value is dynamic. An em is equal to the size of the font that applies to the parent of the elemnt in question. If you haven't set the font size anywhere on the page, then it is browser default which is probably 16px. So by default, 1 em = 16px and 2em =32px.

Font-Weight - Change the thickness of the font. You have a couple of options such as normal, bold, lighter, bolder or numeric value ranging from 100 - 900. 400 weight = normal font weight.

Line-Height - sets the amount of space used for lines such as in text. Kind of like line spacing in Microsoft Word.

Text-Align - controls how the given text is aligned in its parent block element.

Text-Decoration - gives the text an underline, line-through or overline effect

The Box Model - Padding, Margin, Border

Padding
- Padding properties are used to generate space around an element's content, inside of any defined borders (top, right, bottom, left)
Margin
- Margin properties are used to create space around elements, outside of any defined borders (top, right, bottom, left)
